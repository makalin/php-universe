# Data Processor Example

Demonstrates data processing using utility functions from the `utils/` directory.

## Features

- Array processing and transformation
- Data filtering and aggregation
- Statistical calculations
- Logging integration

## Usage

```bash
cd examples/data-processor
composer install
px build --target=native
./dist/data-processor
```

## What It Demonstrates

- Using utility functions (`CommonFunctions`, `Logger`)
- Data processing patterns
- Logging in compiled applications
- Cross-platform data manipulation

## Files

- `src/DataProcessor.php` - Main data processing class
- `universe.toml` - Build configuration
- `composer.json` - Dependencies
